loopeeJP

概要

都道府県を暗記するためのソフトです。

使い方

  1. 「対象」と「モード」を選択し「PLAY」ボタンをクリックします。
  2. 都道府県コードが表示されるので、その都道府県名を入力します。正解するか制限時間になると次の問題が表示されます。
  3. 出題が終わるまで2をくり返します。

loopeeJP

Overview

It is software for memorizing prefectures.

How to use

  1. Select “Target” and “Mode” and click the “PLAY” button.
  2. The prefecture code will be displayed, so enter the prefecture name. If you answer correctly or the time limit is reached, the following questions will be displayed.
  3. Repeat 2 until the question is finished.

Change log

2022/01/20: hotfixes-1.0.1.0

  • Fixed a bug that some items at the bottom of the answering screen disappeared while entering characters.
  • Partially adjusted the display position of markers indicating prefectures

2022/01/20: hotfixes-1.0.2.0

  • Addition of “REPLY” and “BACK” buttons
  • Partially adjusted the display position of markers indicating prefectures
  • Disable menu and game screen cache
  • Change the display position of the rank

2022/01/25: hotfixes-1.0.3.0

  • Changed the app name to “loopeeJP”
  • Abolition of the “DATA” button, which is a list of questions and answers
  • Enlargement of various menu selection button screens
  • Clarify the stacking order and drawing method of maps on the screen during the game

Operating environment

  • Works only when connected to the internet.
  • Android version: Works with 8.0.0 and above.

Input result criteria

  • GREAT: Within 5 seconds: 2 points
  • GOOD: Within 10 seconds: 1 point
  • BAD: Within 15 seconds: 0 points
  • POOR: Over 15 seconds: 0 points

Rank Criteria

  • AAA 8/9 and above
  • AA 7/9 and above
  • A 6/9 or above
  • B 5/9 or above
  • C 4/9 and above
  • D 3/9 and above
  • E 2/9 or above
  • Less than F 2/9

Reference

  • The gameplay (judgment criteria, score rank concept, etc.) is based on the music game “beatmania IIDX” of “KONAMI”.
    [[Official] beatmania IIDX] (https://p.eagate.573.jp/game/2dx/29/)

  • For the design (background color, character font color, button), I referred to Twitter’s dark mode.
    [[Official] twitter] (https://twitter.com)

  • If you get the rank “AAA” in the target “ALL” mode “NORMAL”, you can say that you have memorized all the relationships between “name” and “position” of all prefectures.

  • If you get the rank “AAA” in the target “ALL” mode “HARD”, you can say that you have memorized all the “codes” and “names” of all prefectures.
